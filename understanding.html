The Zero-Knowledge Architecture

This system ensures data is encrypted on your device *before* it ever reaches the cloud. The cloud provider only ever stores scrambled, unreadable data, giving you complete control and confidentiality.

Under the Hood: The Technology Stack

A combination of modern, secure, and browser-native technologies powers the entire system, ensuring performance and reliability without requiring users to install any software.

This bar chart illustrates the key technology categories and their role within the architecture. The frontend handles all user interaction and cryptographic operations, while cloud APIs and backend services are used purely for storage.

The Two-Key System

A powerful two-key mechanism is used. A strong, random key encrypts the file, and a separate key, derived from the user's memorable phrase, encrypts only the file key. This provides both maximum security and user-friendly recovery.
